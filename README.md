# <img src="assets/app.ico" width="48" valign="middle"> Microsoft Activation Tool (MAT)

A simple PowerShell-based application to activate Microsoft Windows and Microsoft Office licenses. The tool provides a clean interface with buttons to trigger activation tasks and displays status updates upon completion.

## 🌟 Features

- One-click activation for Windows and MS Office.
- Displays real-time status messages.
- Lightweight and easy to run on Windows.
- Built entirely with PowerShell.
- Utilizes modern activation methods.

## 🛠️ Requirements

- Windows 10 or 11
- PowerShell 5.1 or higher
- Internet connection (for initial activation)
